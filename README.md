## Installation

Clone the repository:
```bash
git clone https://github.com/aniketr777/Quick-Ai.git
cd Quick-Ai
```
Install dependencies (make sure you have Python and pip installed):
```bash
pip install -r requirements.txt
```

## Usage

Example of running the main script:
```bash
python main.py
```

Refer to documentation or use `--help` for command-line options:
```bash
python main.py --help
```
